## 🧠 About The Project

🚀 A complete **end-to-end Machine Learning project** that predicts whether a flight will be delayed using historical US flight data.

This project covers:
- Data preprocessing
- Feature engineering
- Model training (Deep Learning + ML)
- Data visualization
- Deployment-ready API
